# 💰 Gestor de Presupuesto Personal

Una aplicación web simple para controlar tus ingresos y gastos mensuales. Registra tu sueldo, agrega gastos por categoría y ve exactamente cuánto dinero te queda disponible.

## ✨ Características

- 📊 **Dashboard en Tiempo Real**: Visualiza ingresos, gastos y dinero disponible
- 💾 **Almacenamiento Local**: Tus datos se guardan en el navegador (sin servidor)
- 📅 **Organización por Fecha**: Registra transacciones con fechas específicas
- 🏷️ **Categorías Personalizadas**: Salario, Comida, Transporte, Servicios, Entretenimiento, Salud
- 📈 **Análisis de Gastos**: Ve cuánto gastas por categoría con gráficas
- 📱 **Diseño Responsive**: Funciona perfectamente en móvil y desktop
- 🎨 **Interfaz Moderna**: Diseño limpio y fácil de usar

## 🚀 Cómo Usar

1. **Accede a la aplicación** en el enlace de GitHub Pages (abajo)
2. **Agrega tu sueldo** al principio del mes como "Ingreso"
3. **Registra tus gastos** conforme los hagas
4. **Visualiza** cuánto dinero te queda disponible
5. Tus datos se guardan automáticamente en tu navegador

## 📝 Ejemplo de Uso

```
Ingresos del mes: S/. 1500.00
Gastos:
- Comida (Supermercado): S/. 250.00
- Transporte (Taxi): S/. 80.00
- Servicios (Netflix): S/. 15.00

Dinero Disponible: S/. 1155.00 ✅
```

## 📦 Tecnología

- **HTML5 + CSS3**: Interfaz moderna y responsive
- **JavaScript Puro**: Sin dependencias externas
- **LocalStorage**: Almacenamiento local en el navegador
- **Lucide Icons**: Iconos modernos

## 💡 Privacidad

✅ **Tus datos son 100% privados**
- No hay servidor que almacene tu información
- Todo se guarda en tu navegador local
- Puedes eliminar datos en cualquier momento

## 📋 Cómo Instalar en tu Equipo

Si quieres ejecutarla localmente:

```bash
git clone https://github.com/tu-usuario/Notificador_Cuentas.git
cd Notificador_Cuentas
# Abre index-simple.html en tu navegador
```

- Agregar más categorías
- Modificar moneda (busca "S/." en el código)
- Adaptarlo a tus necesidades
